# دليل تحويل نظام "ريبلا كيدز" إلى تطبيق سطح مكتب مستقل (EXE) 💻🚀

مرحباً بك! لقد قمنا بتهيئة وتجهيز الكود المصدري بالكامل وتزويده بـ **Electron** و **Electron Builder**، ليكون جاهزاً بنسبة 100% للتحويل والترجمة إلى تطبيق ويندوز مستقل بصيغة تنصيبية ممتازة (`.exe`).

يتكون التطبيق من واجهة رسومية تفاعلية حديثة مبنية بـ **React** ومكثفة مع خادم محلي مدمج بـ **Express** لمعالجة الميزات الذكية والاتصال بـ **Gemini API** بشكل متكامل وآمن.

---

## 🛠️ متطلبات التشغيل والترجمة على جهازك المحلي

قبل البدء في جهاز الكمبيوتر الخاص بك، تأكد من تثبيت الأدوات التالية:

1. **برنامج Node.js** (إصدار v18 أو أحدث) -> [تحميل من هنا](https://nodejs.org/)
2. **برنامج Git** (إذا أردت سحب الكود من مستودعك)
3. **محرر الأكواد VS Code (Visual Studio Code)**

---

## 📖 خطوات بناء التطبيق وتشغيله في VS Code خطوة بخطوة

### 1️⃣ الخطوة الأولى: تنزيل الكود وفتحه في VS Code
قم بتنزيل الكود المصدري المضغوط (ZIP) من منصة AI Studio وفك الضغط عنه، ثم افتحه عن طريق VS Code:
- افتح VS Code.
- اختر `File` -> `Open Folder` وحدد مجلد المشروع المنبثق.

### 2️⃣ الخطوة الثانية: تثبيت الحزم والمكتبات (Dependencies)
افتح مبوبة سطر الأوامر (Terminal) في VS Code بالضغط على الاختصار `` Ctrl + ` `` أو من القائمة العليا `Terminal -> New Terminal` ثم اكتب الأمر التالي لتثبيت جميع المكتبات والاعتماديات بما فيها Electron:
```bash
npm install
```

### 3️⃣ الخطوة الثالثة: ضبط مفتاح الذكاء الاصطناعي (Gemini API Key)
إذا كنت ترغب بالاستفادة من ميزة المستشار المحاسبي الذكي محلياً على جهازك:
- أنشئ ملفًا جديدًا باسم `.env` في المجلد الرئيسي للمشروع.
- اكتب بداخل الملف المعرّف الخاص بك على الشكل التالي:
  ```env
  GEMINI_API_KEY=your_actual_gemini_api_key_here
  ```
> **ملاحظة:** سيقوم الخادم المحلي بالتقاط المفتاح تلقائياً وتشغيل التقارير بذكاء كامل!

## 🚀 تشغيل التطبيق في بيئة التطوير (إصدار تجريبي)

لتشغيل التطبيق والتحقق من جودته على سطح المكتب مباشرة قبل الترجمة النهائية، اكتب الأمر التالي في Terminal:

```bash
# الخطوة الأولى: بناء ملفات الواجهة الأمامية والخادم مبدئياً
npm run build

# الخطوة الثانية: تشغيل التطبيق محلياً داخل نافذة Electron
npm run electron:start
```
ستظهر لك نافذة لسطح المكتب تحمل اسم **"ريبلا كيدز"** بداخلها النظام بأكمله وبكامل أداءه اللوجستي والمستودعي!

---

## 📦 إنتاج وتصدير ملف التنصيب النهائي الممتاز (Build EXE File)

لطرح التطبيق للعملاء والموظفين على شكل ملف تثبيت كلاسيكي لويندوز (`.exe`):

1. اكتب في टर्मिनल الأمر التالي:
   ```bash
   npm run electron:pack
   ```
2. انتظر بضع ثوانٍ وسيقوم نظام الترجمة بـ:
   - تحسين الكود وتصغير حجمه ليكون سريع التشغيل للغاية.
   - تهيئة خدمة الخلفية التلقائية.
   - جمع كل الأصول، الأيقونات، الواجهات، وسكربتات الاتصال.
   - إنتاج مجلد جديد بالكامل يسمى `release/`.

3. اذهب إلى المجلد الجديد `release` وستجد بداخله ملف التنصيب الاحترافي:
   - **`RebilaKids Setup 0.0.0.exe`** أو ما يشبهه.

بمجرد النقر عليه مرتين على أي جهاز كمبيوتر يعمل بنظام التشغيل ويندوز، سيتم تثبيت نظام "ريبلا كيدز" وتوفيره على سطح المكتب وفي قائمة "ابدأ" كأيقونة تفاعلية ممتازة!
